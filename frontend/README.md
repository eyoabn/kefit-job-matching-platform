# Kefit Freelance Marketplace - Frontend

## Quick Start
```bash
npm install
npm run dev
```
Frontend runs at: http://localhost:3000

## Environment
Create .env.local:
```bash
VITE_API_URL=http://localhost:8000/api/v1
```
